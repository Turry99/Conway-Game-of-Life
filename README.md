# Conway-Game-of-Life
Conway's Game of Life in C++

To compile on Linux use : " g++ -o GameOfLife main.cpp -lsfml-graphics -lsfml-window -lsfml-system" <br>
<b>Keys:</b> <br>
Enter -->Generate random pattern <br>
Space -->Start <br>
Esc   -->Clear the screen <br>
